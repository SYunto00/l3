# L3 デザインノート

## /about-colors インタラクション構想（保留・後で実装）

チェックボックスではなく「体験」として色を混ぜる。

### PC版
- 画面上部〜中央に**色の丸（7つのテーマ色の球）**がパレットとして並ぶ
- ユーザーは色の丸を**ドラッグして下の「混ぜる器」にポンポン投入**
- 器の中で色が加法混色で合成されていく様子が視覚化される
- 合成色が完成したら、その色のボタンを押すと**該当作品ページへ遷移**

### モバイル版
- スワイプで色の丸を拾う → 下のミキシングエリアにドロップ
- 「ポンポン」と連続投入する感覚
- 合成色タップで遷移

### 体験の核
- 「思想を混ぜる」を身体動作として感じさせる
- チェックで分類するのではなく、手で混ぜる
- 色が完成する瞬間 = 作品との出会いの瞬間

### 実装メモ
- ドラッグ＆ドロップ（PC）+ touch event（モバイル）
- 色合成は既存の加法混色RGBロジックを共用
- 混ぜる器は円形コンテナ、中で色が波打つアニメがあると詩的
