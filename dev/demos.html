和紙のにじみ、色の浮かび上がり、ホバーの間合い — 9案の比較

hover / scroll して体感してください

01 — 和紙のにじみ

α 閾値 / Goo Threshold

SVG blur→feFuncA discrete thresholdで不規則な墨にじみ境界を生成

β 繊維 / Fiber Bias

非等方feTurbulence二層 + feDisplacementMapで繊維方向バイアスのにじみを表現

γ 手描 / Hand-drawn

有機的な凸凹パスをclipPathとして使用し、手描きの墨広がり輪郭を再現

02 — 色の浮かび上がり

α Gooeyブルーム
作品タイトル
bloom / gooey

Gooeyフィルタでチップとパネルの縁が融合しながら450msで展開

β Clip-path+焦点
作品タイトル
clip / blur-resolve

circle clip-path拡張 + blur解消アニメーションで焦点が合うように出現

γ Canvas摂動
作品タイトル
canvas / noise

Canvasに放射状グラデーションをノイズ摂動しながら描画し、有機的に色が広がる

03 — ホバーの間合い

試してみて:
1) マウスを3つのグリッドの上を ゆっくりと スライド → α は全部光る / γ はほぼ光らない
2) マウスをひとつのチップで 止める → 3つすべて光る（γは少し遅い）
3) マウスを 素早く 横切る → α だけ光る
4) γグリッドにマウスを乗せたまま、「試しにスクロール」を押す → γは光らない（スクロール抑制が効いている）
α 即時
FIRES: 0

mouseenterで即時発火。カーソル通過でもすべて反応するノイジーな基準動作

β Delay
FIRES: 0

300ms遅延オープン + 200msクローズ遅延。通過時のノイズを軽減する基本策

γ フル処方箋
FIRES: 0

rest150ms + 300ms上限 + 200msクローズ + スクロール中pointer-events抑制の完全実装
